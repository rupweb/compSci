In SQL, you can retrieve data from multiple tables using the JOIN operation. Here's an example query that retrieves data from two tables: "Orders" and "Customers."

Example SQL Query

        
SELECT Orders.OrderID, Customers.CustomerName, Orders.OrderDate
FROM Orders
INNER JOIN Customers ON Orders.CustomerID = Customers.CustomerID;

This query selects data from the "Orders" table and the "Customers" table and joins them using the "CustomerID" column. It retrieves the "OrderID," "CustomerName," and "OrderDate" columns from both tables for matching records.

Explanation

The query uses the following SQL components:

  • SELECT: Specifies the columns to retrieve.
  • FROM: Specifies the tables to retrieve data from ("Orders" and "Customers").
  • INNER JOIN: Performs an inner join operation, matching records with the same "CustomerID" in both tables.
  • ON: Specifies the condition for the join, matching records based on the "CustomerID" column.

The result of this query will include data from both tables, combining information about orders and customer names for matching records.
