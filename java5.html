Inheritance is a core concept in object-oriented programming that allows one class (known as a subclass or child class) to inherit the attributes and methods of another class (referred to as a superclass or parent class). This mechanism promotes code reusability and establishes a hierarchical relationship between classes.

Key Features of Inheritance:

  • Inheritance supports the concept of “is-a” relationship.
  • It enhances code reusability and readability.
  • Subclasses can add their own methods and variables in addition to the inherited ones.
  • Subclasses can also override inherited methods to provide specific implementation.

Example of Inheritance in Java:

Consider a base class Vehicle and a derived class Car that inherits from Vehicle:


public class Vehicle {
    protected String brand;

    public void honk() {
        System.out.println("Beep beep!");
    }
}

public class Car extends Vehicle {
    private int numberOfDoors;

    public void display() {
        System.out.println("Brand: " + brand + ", Doors: " + numberOfDoors);
    }
}

In this example, Car inherits from Vehicle. It means that Car can use the properties and methods of Vehicle, like the brand field and the honk method. The Car class also adds its own property (numberOfDoors) and method (display).
