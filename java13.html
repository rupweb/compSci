The enhanced for loop, also known as the for-each loop, is a convenient and concise way to iterate through elements in a collection or an array in Java. It simplifies the process of iterating and eliminates the need for explicit indexing.

Syntax of the Enhanced For Loop:


for (elementType element : collection) {
// Code to be executed for each element
}

Where:

  • elementType: The data type of elements in the collection.
  • element: A variable that represents each element in the collection during each iteration.
  • collection: The collection (or array) to be iterated through.

How the Enhanced For Loop Works with Collections:

  • The enhanced for loop iterates through each element in the collection sequentially.
  • During each iteration, the current element is assigned to the variable specified in the loop header (e.g., element in the syntax).
  • The loop executes the specified code block for each element in the collection.
  • Once all elements have been processed, the loop terminates automatically.

Benefits of Using the Enhanced For Loop:

  • It simplifies the code and eliminates the need for explicit indexing or iterators.
  • It reduces the chances of off-by-one errors when iterating through collections.
  • It enhances code readability by making the iteration process more intuitive.

Example with ArrayList:


import java.util.ArrayList;
import java.util.List;

public class ForEachExample {
public static void main(String[] args) {
    List<String> fruits = new ArrayList<>();
    fruits.add("Apple");
    fruits.add("Banana");
    fruits.add("Cherry");

    // Using the enhanced for loop to iterate through the list
    for (String fruit : fruits) {
        System.out.println("Fruit: " + fruit);
    }
}
}

In this example, an `ArrayList` is used to store a list of fruits. The enhanced for loop is used to iterate through the list and print each fruit's name.
